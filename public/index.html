🚗 API de Autos

Esta API permite gestionar autos usando Firebase Firestore y está protegida por autenticación JWT.

🔐 Login

Usá este endpoint para autenticarte y obtener un token JWT:

POST /auth/login

{
  "username": "admin",
  "password": "1234"
}

Respuesta:

{
  "token": "tu_token_jwt"
}

📦 Endpoints de Autos

  • GET /api/autos - Obtener todos los autos
  • GET /api/autos/:id - Obtener auto por ID
  • POST /api/autos/create - Crear un auto
  • DELETE /api/autos/:id - Eliminar un auto

✅ Formato válido de auto

Este es el JSON esperado al crear un auto:

{
  "marca": "Ford",     
  "modelo": "Focus",
  "precio": 12000000,
  "color": "Azul",
  "anio": 2023
}
